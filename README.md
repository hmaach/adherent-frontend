# Adherent - Service Marketplace Platform

A full-stack service marketplace platform developed in collaboration with **Sobol Digital** association, where people with skills can create profiles and publish services they offer.

## Tech Stack

### Frontend

- **React** - Frontend framework
- **Material-UI** - UI component library
- **Redux Toolkit** - State management
- **React Router** - Navigation

### Backend

- **Laravel** - PHP backend framework
- **MySQL** - Database

## Project Structure

This repository contains the **Frontend** application. The Backend is in a separate repository.

### Related Repositories

- **Frontend**: https://github.com/hmaach/adherent-frontend
- **Backend**: https://github.com/hmaach/adherent-backend

## Features

- User authentication and profiles
- Service listings and announcements
- Event management
- Professional sector (Filiere) categorization
- Member directory with search and filtering
- Archive system for documents
- Calendar with events
- Admin panel for platform management

## Demo Mode

The frontend includes a **demo mode** that works without a backend connection. Mock data is used to simulate a fully functional platform.

## Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

```bash
# Install dependencies
npm install

# Start development server
npm start
```

The app will open at [http://localhost:3000](http://localhost:3000).

### Build for Production

```bash
npm run build
```
